## Code

You can write code in a line, such as `echo -e "\033[32mHello"` or `lolololol`, or in a code block:

```python
from flask import Flask
app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Hello World!'

if __name__ == '__main__':
    app.run()
```
